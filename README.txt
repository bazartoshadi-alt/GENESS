Hirova Talent Intelligence OS - Public Ready Static Version

طريقة النشر:
1) فك ضغط الملف.
2) ارفع الملفات التالية مباشرة على GitHub Pages أو أي استضافة Static:
   index.html
   styles.css
   app.js
   README.txt
3) لا ترفع المجلد الداخلي كاملاً إذا كان سيجعل index.html داخل مجلد فرعي.

ملاحظات مهمة:
- الواجهة العامة تظهر للزوار مباشرة.
- مساعد الزوار لا يعرض أي خيار إداري.
- لوحة الإدارة تظهر فقط بعد الدخول.
- كلمة مرور الإدارة غير معروضة داخل واجهة الدخول ولا داخل هذه الملفات كنص واضح.
- البيانات محفوظة داخل المتصفح LocalStorage.

المزايا:
- بوابة شواغر عامة جاهزة للمتقدمين.
- إدارة طلبات الشواغر ونشر الإعلان بزر تشغيل.
- إضافة وحفظ المرشحين.
- Pipeline للمرشحين.
- مركز ذكاء لصاحب الموقع بعد الدخول.
- مساعد متقدمين عائم أسفل الموقع.
- تصدير واستيراد نسخ احتياطية.
